Quick Install

macOS

  1. Download ZHDownloader-macOS.pkg from the ZH Motions student link.
  2. Double-click the .pkg file.
  3. Click Continue → Continue → Install → enter Mac password → Done.
  4. First launch: open Applications, right-click "ZH Downloader" → Open → Open again (one-time security bypass).
  5. App launches. Ready.

Windows

  1. Download ZHDownloader-Setup.msi from the ZH Motions student link.
  2. Double-click the .msi file.
  3. Follow installer prompts.
  4. Launch from Start Menu → ZH Downloader.
  5. If Windows shows "unrecognized app" warning: More info → Run anyway.

First-Time Setup

  1. Settings tab:
    • Theme: Light (default) or any from 6 options
    • Concurrent downloads: 2 (default) or higher
  2. Downloads tab:
    • Cookies dropdown → select chrome
    • Format → 4K (2160p) or HD (1080p)
  3. Browser: log in to YouTube / Artgrid / Artlist in Chrome for HD downloads.

How to Use

Method 1 — Paste URL

  1. Copy a video URL (YouTube, Vimeo, TikTok, Artgrid, etc.)
  2. Paste into "Paste URLs" box (multiple URLs OK, one per line)
  3. Click ↓ Download
  4. File saved to ~/Downloads/ZHDownloader/

Method 2 — Browser Extension

  1. Install Chrome extension (see below)
  2. Visit video page → floating button bottom-right
  3. Click button → URL auto-sent to app → starts downloading

Method 3 — Clipboard Auto-Detect

  • Enable "Watch clipboard" checkbox
  • Copy any video URL → app auto-adds to queue

Install Browser Extension

Works with Chrome, Edge, Brave.

  1. Open chrome://extensions/
  2. Enable Developer mode (top right)
  3. Click Load unpacked
  4. Select the extension/ folder
  5. Pin the ZH icon to toolbar

Premiere Pro Compatibility

HD and 4K formats auto-transcode to H.264 (avc1) + AAC in MP4.
Drag the file directly into Premiere timeline — opens without re-encoding.
