## 🚀 Quick start

### Step 1 — Download installer

Go to: **[Latest Release](https://github.com/zhmotionspanel-cmyk/ZHDownloader-v4/releases/latest)**

| Platform | File to download |
|----------|------------------|
| 🍎 **Mac** | `ZHDownloader-macOS.pkg` |
| 🪟 **Windows** | `ZHDownloader-Setup.msi` |

### Step 2 — Install

Open the downloaded file → follow installer prompts.

**Mac users:** After install, right-click app in Applications → **Open** → **Open Anyway** (one-time security bypass).

### Step 3 — Read the install guide

Open **`INSTALL-STUDENTS.md`** or **`INSTALL-STUDENTS.html`** in this folder for:
- First-time setup (Cookies + Format)
- How to download (3 methods)
- Browser extension install
- Premiere Pro compatibility
- Troubleshooting

### Step 4 — Read the license

Open **`LICENSE-STUDENT.txt`** — important terms about usage.
